# TasselDistanceMatrix Class

Class `TasselDistanceMatrix` defines a `rTASSEL` Class for storing
TASSEL genotype and phenotype objects.

Prints out information related taxa, positions, genotype, and phenotype
information.

## Usage

``` r
# S4 method for class 'TasselDistanceMatrix'
show(object)
```

## Arguments

- object:

  a `TasselGenotypePhenotype` class object

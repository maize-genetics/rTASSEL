# Return `rJava` reference object

Returns the `rJava` memory reference for a given `rTASSEL` object

## Usage

``` r
javaRefObj(object, ...)

# S4 method for class 'TasselGenotype'
javaRefObj(object)

# S4 method for class 'TasselNumericGenotype'
javaRefObj(object)

# S4 method for class 'TasselPhenotype'
javaRefObj(object)
```

## Arguments

- object:

  an `rTASSEL` object

- ...:

  Additional arguments, for use in specific methods
